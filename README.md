# umux

Windows-first Rust port/recode of cmux.

`umux` is designed around a workspace-first workflow with terminal and browser surfaces, split panes, notification-driven unread navigation, session restore, and a local CLI/IPC control surface.

The `cmux/` directory in this workspace is a GPL-licensed reference checkout. New Rust implementation work lives outside that directory.
